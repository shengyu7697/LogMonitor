LogMonitor
==========

## Introduction
LogMonitor is a tool that can keep printing log from file.

## System Requirement
Windows 7, XP, Linux, MacOS

## How to Build
### Windows
use Visual studio (Visual studio 2013 example)
```
mkdir -p build
cd build
cmake .. -G "Visual Studio 12 2013"
```
open the *.sln file and build

### Linux, MaxOS
use Makefile
```
make
```

or use CMake
```
mkdir -p build
cd build
cmake ..
make
```
